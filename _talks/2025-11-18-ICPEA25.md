## Abstract

This talk presents a double-loop Active Disturbance Rejection Control (ADRC) scheme for Push-Pull DC-DC converters. The converter average model is decomposed into two disturbed integrator subsystems — an inner current loop and an outer voltage loop — each controlled by a first-order ADRC with Extended State Observer (ESO) for real-time disturbance estimation and cancellation. Simulation results demonstrate robust voltage regulation and fast tracking under load and input voltage variations.

## Outline
- System Description and Modeling
- Fundamentals of First-Order ADRC
- Double-Loop ADRC Architecture
- Simulation Results and Performance Evaluation
